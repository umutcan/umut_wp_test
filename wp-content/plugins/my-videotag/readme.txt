== Installation ==

= Minimal Requirements =
* PHP: 5.x.x
* WordPress: 3.0

= Recommended Requirements =
* PHP: 5.2.x or newer
* WordPress: 3.0.1 or newer


= ---------- ENGLISH ---------- =

= Basic Installation =

 - 1. Unpack the downloaded zipped file
2. Upload the "my-videotag" folder to your /wp-content/plugins directory
3. Log into Wordpress
4. Go to the "Plugins" page
5. Activate the My Videotag Plugin
6. Use the `[video]` shortcode in your post, page or comments.

= IMPORTANT: =

If you experience any problems remember the following things:

 - 1. For Tu.tv you need `allow_url_fopen` enabled.
2. Websites may change really rapidly so it may be a change in how sites broadcast videos.
3. Sometimes servers don't have enough bandwidth to connect to video servers and they fail.
4. Loading too many videos in the same page can be quite annoying for the user and some browsers don't handle very well too many flash players at the same time.

== Frequently Asked Questions ==

= Using the plugin =

* Basic syntax: `[video] LINK VIDEO [/video]`
* Advanced syntax: `[video w="pixel size" h="pixel size"] LINK VIDEO [/video]`
* Example: `[video w="450" h="300"]LINK VIDEO[/video]`


= ---------- ENGLISH ---------- =


= How to use video links break.com =

* The plugin only works with links like this: `http://www.break.com/index/title-of-video-2143025`
* If the link is like this `http://www.break.com/index/title-of-video-not-id.html`
* must find the video id in embed code, example: **id='539008'**
* And use it this way: `http://www.break.com/index/title-of-video-not-id-539008`

= Why doesn't this plugin support such-and-such site? =

There are few possible reasons for this:

* I may have never heard of the site or simply not been mentioned on [my post plugin](http://toxigeek.com/plugin-my-videotag) may make me include it in a future release.
* The URL at which the video can be viewed has nothing in common with the embed URL. This means my plugin can't do anything with the URL you give it. Support for fetching the emded URL from the website may be added in a future version though, we'll see.
* I have deemed the site not popular enough to warrant being added to my plugin. I don't wish to bloat my plugin with tiny little sites that only one or two people will use.

= My-Videotag plugin is compatible with the plugin "Viper's Video Quicktags"? =

* My-Videotag is incompatible with plugins that use the shortcode "VIDEO"
* Solution to use "Viper's Video Quicktags"
* Remove `add_shortcode( 'video', array(&$this, 'shortcode_videofile') ); // Legacy`
in the file = Vipers-video-quicktags.php**


= My-Videotag is compatible with the plugins "Disqus Comment System" or "Intense Debate" =

* My-Videotag is not compatible
* But disable "*Enable/Disable display videos in comments*" plugins work
