== Description ==

Live Chat is the easy way to have a live chat in your your wordpress blog.

For more information, check out <a href="http://www.danycode.com/live-chat/">Live Chat</a>

Plugin Features:

* Users can use the live chat with or without registration

* Easy to install, no configuration is required

* The Live Chat is implemented with JQuery and Ajax technology

* Customizable colors

== Installation ==

1. Upload the Live Chat plugin to your blog and activate it. 
2. You can see the live chat on the front end of your wordpress website.
